## Get Udis and Dollars from Banxico

### Prerequisites

- Python 3.4
- MySQL 8.0.22

### Getting Started

1.- Clone this repo

2.- Create .env file at the root project

### Secrets examples

```
# Django app key
SECRET_KEY='django_app_secret'

ENGINE='django.db.backends.mysql'
DB_NAME='your_db'
USER='your_user'
PASSWORD='your_password'
HOST='host'
PORT='port'

# Banxico token
Q_TOKEN='your_token'

BASE_URL='url...'
END_POINT='end_point...'
```
